# Android-Mobile-Programming-Final-Work
移动终端程序设计课程设计

## 注意 ##
使用的编程语言是：***Kotlin***，结构使用的是：***Jetpack Compose***

实现了两个功能：**备忘录** 和 **ToDoLists**

备忘录除了可以进行增删改查外，还可以进行收藏

ToDoLists只能进行增删改查，同时删除操作通过将选中的List向左或右滑出屏幕即可删除
